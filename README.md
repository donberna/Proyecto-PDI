# Proyecto-PDI
Para una imagen sin ruido    --n

Para una imagen con ruido    --r

    ./Imagen [option] <image.pgm>
    
    
##Ejemplo:
Entrada:

    ./Imagen --n ../../imagenes/In/RLA4.pgm
    
    
Salida:

    Imagen sin ruido: ../../imagenes/In/RLA4.pgm
    Diametro externo: 203
    Diametro interno: 148
    Espesor: 31
